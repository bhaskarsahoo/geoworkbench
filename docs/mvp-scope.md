# MVP Scope

## MVP Objective

Create a usable borehole interpretation workspace that demonstrates:

- geological visualization
- data ingestion
- human-in-the-loop interpretation
- AI-assisted workflows
- future platform potential

The MVP does not need advanced AI initially.

## Primary Workflow

1. Import borehole data
2. Visualize boreholes and curves
3. Review lithology intervals
4. Edit and correct intervals
5. Suggest seam correlations
6. Generate summaries and exports

## Core MVP Modules

### 1. Data Import Module

Supported formats:

- CSV
- Excel
- LAS

Input categories:

- borehole collar data
- interval lithology
- geophysical logs
- seam information
- coal quality

### 2. Borehole Visualization Module

Features:

- lithology columns
- geophysical curves
- zoom and pan
- multiple borehole comparison
- interval editing
- seam highlighting

### 3. Data Validation Module

Checks:

- missing intervals
- overlapping intervals
- invalid depth ranges
- inconsistent naming
- seam continuity issues

### 4. AI Suggestion Layer

Initial AI features:

- lithology suggestions
- seam matching suggestions
- anomaly flagging
- draft report generation

### 5. Geological Assistant

Natural-language assistant for:

- summaries
- geological observations
- querying datasets
- report drafting

### 6. Export and Reporting

Exports:

- CSV
- Excel
- PDF summaries
- section-ready datasets

## Non-Goals for MVP

Do not initially build:

- advanced 3D geological modeling
- mine planning systems
- real-time drilling systems
- enterprise deployment complexity
- highly customized ML training infrastructure

## Success Criteria

The MVP succeeds if geologists say:

- this saves time
- this improves consistency
- this reduces repetitive work
- this is easier than current workflows
- this could become part of daily work
