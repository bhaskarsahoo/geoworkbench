# GeoWorkbench Wiki

This wiki is the long-form documentation home for the production borehole correction system.

## Sections

- `user-manual/` - field user and central geologist workflows.
- `admin-manual/` - project setup, user roles, dictionaries, layouts, and templates.
- `architecture/` - system architecture, module boundaries, data flow, and deployment architecture.
- `data-model/` - canonical schemas, versioning, audit, display settings, and import/export mappings.
- `api/` - API reference, generated OpenAPI notes, webhooks, and integration contracts.
- `deployment/` - local, on-prem, cloud, backup, monitoring, and troubleshooting guides.

## Documentation Principle

Every milestone should update this wiki as part of delivery. The system should be explainable to geologists, administrators, IT teams, and future developers without relying on verbal handover.
