# Borehole Workbench User Manual

## Runtime Mode

Runtime mode is used by central geologists to review and correct borehole logs.

Current seeded workflow:

1. Select a borehole.
2. Review lithology, seam, curve, and corebox tracks.
3. Click a lithology interval to open depth metadata.
4. Edit lithology, seam, or remarks.
5. Save the correction.
6. Hover over curves to inspect the nearest measured sample.
7. Click a corebox reference to open the full image.

## Edit Display Mode

Edit display mode is planned for configuring widgets, tracks, and curve settings. It should not change geological data.
